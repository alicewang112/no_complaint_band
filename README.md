# 不抱怨手环

一个轻量级的“不抱怨手环”网页应用。用户发现自己抱怨时，把页面中的紫色手环拖到另一只手上，应用会记录本次持续时长，并从当前时间重新开始计时。

## 功能

- 连续不抱怨计时：展示天、小时、分钟、秒。
- 拖拽换手：将手环拖到另一只手后自动重新计时。
- 本地记录：记录每次换手前的持续时长。
- 数据统计：展示当前记录、最长记录、换手次数、平均记录和累计时间。
- 清空数据：底部按钮可清空本地记录并重新开始。
- 移动端优先：适合直接在手机浏览器中使用。

## 运行方式

项目已通过 GitHub Pages 部署，可以直接在浏览器中访问：

```text
https://alicewang112.github.io/no_complaint_band/
```

如果需要在本地开发或预览，也可以启动一个静态服务器：

```bash
python3 -m http.server 4173
```

然后访问：

```text
http://127.0.0.1:4173/
```

也可以直接打开 `index.html` 查看页面，但推荐使用静态服务器预览，行为更接近线上环境。

## 数据存储

应用使用浏览器 `localStorage` 保存数据，不需要登录或后端服务。

存储 key：

```text
no-complaint-band-state-v1
```

主要数据包括：

- `startTime`：当前计时开始时间
- `currentHand`：当前手环所在手
- `history`：历史换手记录

点击“清空数据”会清空历史记录，并将计时起点重置为当前时间。

## 设计原则

- 打开即用，不增加记录负担。
- 交互尽量简单，换手即重新开始。
- 数据只保存在本地，保护隐私。
- 页面保持轻量，不引入框架和构建工具。
